# tutorial-praca-z-gitem

Git jest systemem kontroli zmian w plikach, umożliwia on zapisywanie histori zmian pliku oraz koordynuje prace kilku osób nad projektem. Zapisana historia zmian może być przetrzymywana na serwerze, z którego będziemy mogli zarządzać naszym repozytorium z kodem, przez co będzie on bezpieczny i nie utracimy w skutek awarii systemu.
